Kanchanaburi


กาญจนบุรี เป็นจังหวัดที่มีที่เที่ยวเยอะแยะมากมาย เที่ยวได้ทั้งปี ไม่ว่าจะเป็นหน้าร้อน หน้าฝน หรือหน้าหนาว ก็ชิลได้ไม่มีคำว่าเบื่อแน่นอน

อีกทั้งยังเป็นที่เที่ยวใกล้กรุงเทพฯ ขับรถแป๊ปเดียวถึงอีกด้วย วันนี้เราขอพาไปเช็คลิสต์อัปเดตที่เที่ยวกาญจนบุรี เช็คอินเที่ยวฟินพร้อม ๆ กัน


สถานที่ท่องเที่ยวในจังหวัดกาญจนบุรี

  1. เขาช้างเผือก


    2. ที่เที่ยวกาญจนบุรียอดนิยม ขึ้นเขาไปสัมผัสกับธรรมชาติกันที่เขาช้างเผือก สถานที่ท่องเที่ยวสุดท้าท้ายสำหรับสายลุยๆ เป็นเขาที่ตั้งอยู่ที่อุทยานแห่งชาติทองผาภูมิ มีความสูง 1,249 เมตร และเป็นภูเขาที่สูงเป็นอันดับสามของจังหวัดกาญจนบุรี เปิดให้นักท่องเที่ยวได้ผจญภัยกันช่วงเดือนตุลาคม-กุมภาพันธ์ของทุกปี ต้องติดต่อรับคิวในการขึ้นเขากับเจ้าหน้าที่อุทยานฯก่อนขึ้น วิวสวย บรรยากาศดีมากๆ

    ที่อยู่ : อุทยานแห่งชาติทองผาภูมิ ตำบลปิล๊อก อำเภอทองผาภูมิ จังหวัดกาญจนบุรี

    เปิด-ปิด : เดือนตุลาคม-กุมภาพันธ์ ของทุกปี

    เบอร์โทร : 0-3451-0979

    เว็บไซต์ : -

  2. น้ำตกเอราวัณ


    3. ชุ่มชื่นกันต่อด้วยการเที่ยวน้ำตกเอราวัณ น้ำตกชื่อดังที่พลาดไม่ได้ของจังหวัดกาญจนบุรี เป็นน้ำตกสีเขียวมรกตสดใส ตั้งอยู่ห่างจากที่ทำการอุทยานประมาณ 500 เมตร มีความสูงจากระดับน้ำทะเลตั้งแต่ 100 - 400 เมตร แบ่งออกเป็น 7 ชั้น แต่ละชั้นสวยมากและมีบริเวณให้เล่นน้ำได้ มีน้ำตลอดปี

    ที่อยู่ : อุทยานแห่งชาติเอราวัณ ตำบลท่ากระดาน อำเภอศรีสวัสดิ์ จังหวัดกาญจนบุรี

    เปิด-ปิด : 06:00-18:00 น.

    เบอร์โทร : 0-3451-0979

    เว็บไซต์ : -

  3. ลำคลองงู


    4. อีกหนึ่งที่เที่ยวกาญจนบุรีที่พลาดไม่ได้ของกาญจนบุรีก็คือ ลำคลองงู ใครสายแอดเวนเจอร์ต้องมา ตั้งอยู่ที่ อุทยานแห่งชาติลำคลองงู เป็นที่ท่องเที่ยวธรรมชาติ มีทั้งการเดินถ้ำชมเสาหินที่สูงที่สุดในโลก น้ำตกที่อุดมสมบูรณ์ บรรยากาศดีมากๆ ใกล้ชิดธรรมชาติสุดๆ เปิดให้เที่ยวปีละ 2 เดือนเท่านั้น คือเดือนมีนาคมและเมษายน

    ที่อยู่ : อุทยานแห่งชาติลำคลองงู อำเภอทองผาภูมิ จังหวัดกาญจนบุรี

    เปิด-ปิด : 08:00-16:00 น.

    เบอร์โทร : 084-913-2381

  4. เขื่อนวชิราลงกรณ์


    5. เขื่อนวชิราลงกรณ์เป็นอีกหนึ่งสถานที่ท่องเที่ยวยอดนิยมของจังหวัดกาญจนบุรีที่นักท่องเที่ยวต้องมา ตั้งอยู่อำเภอทองผาภูมิ บรรยากาศดีมากๆ มองเห็นวิววิวทะเลสาบ ที่โอบล้อมไปด้วยภูเขา สามารถขับรถขึ้นไปชมวิวบริเวณสันเขื่อนได้ และมีกิจกรรมให้เล่นไม่ว่าจะเป็นล่องแพ ดูนก ตกปลา

    ที่อยู่ : เขื่อนวชิราลงกรณ์ หมู่ 1 ตำบลท่าขนุน อำเภอทองผาภูมิ จังหวัดกาญจนบุรี

    เปิด-ปิด : 06:00-18:00 น.

    เบอร์โทร : -

    เว็บไซต์ : -

  5. น้ำตกห้วยแม่ขมิ้น


    6. ต่อกันด้วยอีกหนึ่งน้ำตกน่าเที่ยวของกาญจนบุรีอย่าง น้ำตกห้วยแม่ขมิ้น เป็นน้ำตกที่ตั้งอยู่ในพื้นที่อุทยานแห่งชาติเขื่อนศรีนครินทร์ สวยมากๆ มี 7 ชั้นด้วยกัน โดยชั้นแรกถึงชั้น 4 มีระยะทางแค่ 300-750 เมตร ส่วนชั้นที่ 5 ขึ้นไปจนถึงชั้นบนสุ มีระยะทางมากกว่า 1 กิโลเมตร ทิวทัศน์น้ำตกแต่ละชั้นไม่เหมือนกัน ทำให้ได้วิวที่สวยมากๆ

    ที่อยู่ : อุทยานแห่งชาติเขื่อนศรีนครินทร์ ตำบลท่ากระดาน อำเภอศรีสวัสดิ์ จังหวัดกาญจนบุรี

    เปิด-ปิด : 08:00-17:00 น.

    เบอร์โทร : 062-325-5498

    เว็บไซต์ : -

  6. ถ้ำธารลอด


    7. อีกหนึ่งสถานที่ที่เที่ยวกาญจนบุรีก็คือ ถ้ำธารลอด เป็นถ้ำสวย ตั้งอยู่ในอุทยานแห่งชาติเฉลิมรัตนโกสินทร์ เป็นถ้ำที่เกิดจากการยุบตัวของหินปูน และการกัดเซาะของน้ำ ทำให้เขาหินปูนกลายสภาพเป็นสะพานธรรมชาติขนาดใหญ่ มีธารน้ำใสไหลลอดผ่านถ้ำ ภายในถ้ำมีหินงอกหินย้อยสวยงม แปลกกน่าให้เดินชม ใครชอบเที่ยวธรรมชาติต้องไม่พลาด

    ที่อยู่ : อุทยานแห่งชาติเฉลิมรัตนโกสินทร์ ตำบลเขาโจด อำเภอศรีสวัสดิ์ จังหวัดกาญจนบุรี

    เปิด-ปิด : 08:30-16:30 น.

    เบอร์โทร : 095-619-8981

    เว็บไซต์ : -

  7. สังขละบุรี


    8. อีกหนึ่งแลนด์มาร์คที่พลาดไม่ได้ของจังหวัดกาญจนบุรีก็คือ สังขละบุรี เป้นหนึ่งที่ท่องเที่ยวยอดนิยมที่ทุกคนต้องมา เป็นหมู่บ้านและชุมชนมอญเล็กๆ อยู่ใกล้ชายแดนเพื่อนบ้านประเทศเมียนมา ที่เต็มไปด้วยมนต์เสน่ห์ ทั้งควิถีชีวิต ความเรียบง่าย และความสงบ มีกิจจกรรมให้ทำมากมายไม่ว่าจะเป็น ตักบาตรบนสะพานมอญ เดินถ่ายรูปเล่นหรือล่องเรือไปชมน้ำจมน้ำ รวมไปถึงเที่ยวเจดีย์พุทธคยา

    ที่อยู่ : อำเภอสังขละบุรี จังหวัดกาญจนบุรี

    เปิด-ปิด : 06:00-18:00 น.

  8. สกายวอล์คสองแคว


    9. ที่เที่ยวกาญจนบุรีเปิดใหม่ แลนด์มาร์คใหม่ล่าสุดของกาญจนบุรีกับ สกายวอล์คสองแคว เป็นสกายวอล์ค ที่ตั้งอยู่ริมแม่น้ำแควใหญ่ อำเภอเมือง ใกล้กับศาลหลักเมืองกาญจนบุรี เป็นจุดลงเรือสำหรับล่องเรือไปชมแม่น้ำสองสี ที่นี้เป็นจุดชมวิวแม่น้ำสองสีของแม่น้ำแควน้อย และ แม่น้ำแควใหญ่ ไหลมาบรรจบกัน วิวสวยมากๆ ตัวสกายวอร์คทำด้วยกระจกใส มีระยะทาง 150 เมตร สูง 12 เมตร

    ที่อยู่ : บริเวณท่าน้ำหน้าเมืองกาญจนบุรี อำเภอเมืองกาญจนบุรี จังหวัดกาญจนบุรี

  9. อุทยานประวัติเมืองสิงห์


    10. มาเที่ยวกาญจนบุรีทั้งที่จะพลาดเที่ยวอุทยานประวัติศาสตร์เมืองสิงห์ไปได้ยังไง ปราสาทเมืองสิงห์ เป็นปราสาทหินที่ถูกสร้างขึ้นเพื่อเป็นพุทธศาสนสถานในพุทธศาสนา นิกายมหายาน เป็นสถาปัตยกรรมและปฏิมากรรม คล้ายคลึงกับของสมัยพระเจ้าชัยวรมันที่ 7กษัตริย์นักสร้างปราสาทแห่งขอม แวดล้อมด้วยทิวเขาเป็นแนวยาวอยู่โดยรอบ บรรยากาศดี ภายในปราสาทมีป้าย QR Code ให้สแกนอ่านประวัติศาสตร์ และสถานที่สำคัญๆด้วย

  10. ถ้ำกระแซะ ทางรถไฟสายมรณะ


    11. หนึ่งในแลนด์มาร์คที่พลาดไม่ได้ถ้ามาเยือนกายจนบุรีก็คือ ถ้ำกระแซ ทางรถไฟสายมรณะ ตั้งอยู่ติดกับเส้นทางรถไฟสายมรณะอันเป็นหนึ่งในประวัติศาสตร์สมัยสงครามโลกครั้งที่ 2 เป็นถ้ำที่เคยเป็นที่พักของเชลยศึกเมื่อครั้งสร้างเส้นทางรถไฟสายมรณะจากไทยไปพม่า ตัวถ้ำติดกับเส้นทางรถไฟสายกาญจนบุรี-น้ำตก และยังเป้นจุดชมวิวแม่น้ำแควที่สวยที่สุดในกาญจนบุรีด้วย สามารถเดินไปถ่ายรูปเล่นได้บนรางรถไฟได้ แต่ต้องระวังให้มากๆ

    ที่อยู่ : ตำบลลุ่มสุ่ม อำเภอไทรโยค จังหวัดกาญจนบุรี

  11. สะพานข้ามแม่น้ำแคว


    12. ที่เที่ยวกาญจนบุรียอดฮิต สะพานข้ามแม่น้ำแคว เป็นอีกหนึ่งสถานที่ต้องแวะมา เป็นจุดเชื่อมโยงของประวัติศาสตร์โลกในหน้าสงครามโลกครั้งที่ 2 และเป็นอนุสรณ์สงครามมหาเอเชียบูรพา บรยากาศดีมากๆ ใครมแล้วต้องได้มาเดินบนรางรถไฟบนสะพานข้ามแม่น้ำแคว ชมความงามของสองฝากฝั่งแม่น้ำแคว

    ที่อยู่ : ตำบลท่ามะขาม อำเภอเมืองกาญจนบุรี จังหวัดกาญจนบุรี
